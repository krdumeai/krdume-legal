Exclusão de Dados do Usuário

Aplicativo: KRDUME Radar  |  Atualizado: 23/04/2026

Como solicitar a exclusão

O KRDUME Radar não armazena dados pessoais em servidores externos. Todos os dados coletados via API Meta são processados localmente. Para solicitar a exclusão completa:

  1. Envie um e-mail para leandro@krdume.com com o assunto: "Exclusão de Dados KRDUME Radar"
  2. Informe o ID da sua conta Facebook ou página vinculada
  3. Você receberá confirmação em até 72 horas

O que será excluído

  • Tokens de acesso da sua conta
  • Dados de campanhas armazenados localmente
  • Identificadores de páginas e contas de anúncios
  • Logs de uso do aplicativo

Revogar acesso imediatamente

Para remover o acesso sem esperar, acesse:
Facebook → Configurações → Aplicativos e Sites
e remova o KRDUME Radar.

Confirmação

Após a exclusão, você receberá um e-mail de confirmação. O acesso do aplicativo será revogado automaticamente.
