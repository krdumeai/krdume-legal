Política de Privacidade

Aplicativo: KRDUME Radar  |  Versão: 1.0  |  Atualizado: 23/04/2026

1. Sobre o Aplicativo

O KRDUME Radar é um aplicativo de uso interno da KRDUME Ventures para pesquisa de inteligência de mercado. Acessa dados públicos de anúncios via Meta Ads Library API para análise competitiva e identificação de tendências.

2. Dados Coletados

Coletamos apenas:

  • Dados públicos de anúncios disponíveis na Ads Library (nome de página, data de criação, texto do anúncio)
  • Métricas agregadas de campanhas da conta do operador (impressões, alcance, gasto)
  • Identificadores de contas publicitárias e páginas vinculadas pelo usuário

Não coletamos: senhas, dados de cartão de crédito, dados de saúde, dados de menores de idade.

3. Uso das Informações

Os dados são usados exclusivamente para:

  • Análise de tendências de mercado em anúncios brasileiros
  • Geração de relatórios de inteligência competitiva de uso interno
  • Pesquisa de oportunidades de monetização com IA

Não utilizamos os dados para publicidade direcionada, nem os compartilhamos com terceiros.

4. Armazenamento e Retenção

Todos os dados são processados e armazenados localmente no dispositivo do operador. Nenhum dado é enviado a servidores externos além da própria API da Meta. Os dados são excluídos automaticamente após 30 dias.

5. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a: confirmar existência de tratamento, acessar seus dados, corrigi-los, solicitar exclusão e revogar o consentimento a qualquer momento.

Para exercer esses direitos: leandro@krdume.com

6. Exclusão de Dados

Para solicitar a exclusão dos seus dados, acesse nossa página de exclusão de dados ou revogue o acesso diretamente em Facebook → Configurações → Aplicativos.
